Chapter 9 Eleccion del modelo definitivo

A lo largo de este trabajo hemos hecho un analisis exhaustivo de nuestra base de datos relacionado al gdp per capita en colombia, hechos hecho transformaciones a la serie original, la hemos diferenciado y tambien hemos evaluado esta misma serie en distintos modelos y metodos predictivos, ahora lo ultimo es dedicir, cual de todos es el mas indicado para predecir los datos a partir de 2022 en adelante.

Haciendo una recapitulacion:

  • Se ha descartado el modelo Facebook Prophet como modelo ideal debido al inmenso valor en la suma de los cuadrados de los residuos que posee, producto de su notable desviacion luego de la epoca mas critica de nuestros datos (la pandemia)

  • Se han descartado los metodos de prediccion de las redes neuronales de Elman y Jordan debido a la notable dispercion que sus graficas poseen frente a la serie original, indicando a simple vista que sus predicciones tendran variaciones muy marcadas

Entonces a continuacion, compararemos el nivel de prediccion de los modelos de Holt-Winters, el modelo de regresion lineal y el modelo ARIMA para determinar entre los tres cual es el mejor, para esto usaremos la comparativa en el valor RMSE (raiz del error cuadratico medio). Comenzamos con el modelo Holt-Winters

# Dividir los datos en entrenamiento (hasta 2021) y prueba (2022)
entrenamiento <- window(ST_TRIMESTRAL, end = c(2021, 4))
prueba <- window(ST_TRIMESTRAL, start = c(2022, 1))

# Modelo Holt-Winters
modelo_HW <- HoltWinters(entrenamiento)

# Predicciones para los próximos 4 trimestres
predicciones_HW <- forecast(modelo_HW, h = 4)

# Calcular el RMSE
valores_reales <- prueba
prediccion_hw <- predicciones_HW$mean
rmse <- sqrt(mean((valores_reales - prediccion_hw)^2))
cat("El RMSE del modelo Holt-Winters es:", rmse, "\n")
## El RMSE del modelo Holt-Winters es: 11.63841

A continuacion continuamos con el RMSE del modelo de regresion lineal

modelo_LI <- lm(ST_TRIMESTRAL~time(ST_TRIMESTRAL))

# Predicciones del modelo
predicciones_LI <- predict(modelo_LI)

# Valores observados
observados <- ST_TRIMESTRAL

# Cálculo del RMSE
rmse <- sqrt(mean((observados - predicciones_LI)^2))

# Mostrar el RMSE
cat("El RMSE del modelo de regresion lineal es:", rmse, "\n")
## El RMSE del modelo de regresion lineal es: 655.8118

Por ultimo calculamos el RMSE del modelo ARIMA

# Predicciones dentro de la muestra (in-sample)
predicciones_AR <- fitted(modelo_AR) # Valores ajustados por el modelo

# Valores observados
observados <- ST_TRIMESTRAL

# Cálculo del RMSE
rmse <- sqrt(mean((observados - predicciones_AR)^2))

# Mostrar el RMSE
cat("El RMSE del modelo ARIMA es:", rmse, "\n")
## El RMSE del modelo ARIMA es: 54.1927

Al comparar los RMSE entre los tres modelos observamos que los dos que mejor indice de prediccion tienen respecto a los datos reales son el modelo ARIMA y el modelo de Holt-Winters, este ultimo con el menor RMSE de los tres

Para finalizar un grafico que recompila los 3 modelos frente a la serie de tiempo original:

# Modelo 1: Regresión Lineal
modelo_LI <- lm(ST_TRIMESTRAL ~ time(ST_TRIMESTRAL))
predicciones_LI <- ts(predict(modelo_LI), start = start(ST_TRIMESTRAL), frequency = frequency(ST_TRIMESTRAL))

# Modelo 2: ARIMA 
modelo_ARIMA <- auto.arima(ST_TRIMESTRAL)
predicciones_ARIMA <- fitted(modelo_ARIMA)

# Modelo 3: Holt-Winters 
modelo_HW <- HoltWinters(ST_TRIMESTRAL)
predicciones_HW <- fitted(modelo_HW)[, 1]  # Valores ajustados (in-sample)

# Combinar las series in-sample en un único objeto ts
combinado <- cbind(
  Observado = ST_TRIMESTRAL,
  Regresion = predicciones_LI,
  ARIMA = predicciones_ARIMA,
  HW = predicciones_HW
)

# Graficar con dygraph
dygraph(combinado, main = "Comparación de Modelos") %>%
  dySeries("Observado", label = "Serie Original", color = "black") %>%
  dySeries("Regresion", label = "Regresión Lineal", color = "blue") %>%
  dySeries("ARIMA", label = "ARIMA", color = "red") %>%
  dySeries("HW", label = "Holt-Winters", color = "purple") %>%
  dyOptions(stackedGraph = FALSE, drawPoints = TRUE, pointSize = 2) %>%
  dyRangeSelector()  # Agrega un selector de rango interactivo
